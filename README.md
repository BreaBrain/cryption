# cryption
Encrypt files in AES 256-Bit with 3 layers
# Install
```
cd cryption

chmod +x install.sh

./install.sh
```
# Uninstall
```
cd cryption

chmod +x uninstall.sh

./uninstall.sh
```
# Usage
```
encrypt (file/folder) (three numbers between 0 and 9)

decrypt (file/folder) (same three nubmers)
```
# Note
if you delete /etc/encrypt you cant decrypt files!
